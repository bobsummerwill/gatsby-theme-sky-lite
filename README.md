# Gatsby Theme: Sky Lite

A lightweight GatsbyJS theme base with Material-UI and MDX Markdown support.

## Install

```bash
yarn add gatsby-theme-sky-lite # Or, npm install gatsby-theme-sky-lite
```

```js
// gatsby-config.js
module.exports = {
  ...
  plugins: [
		{ resolve: `gatsby-theme-sky-lite` },
		...
	]
  ...
}
```
